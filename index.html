📋 1. Начало работы

  • Загрузка списка студентов: Нажмите "Загрузить список (.txt)" и выберите текстовый файл, где каждая строка - ФИО студента
  • Добавление вручную: Нажмите "Добавить студента" и введите ФИО в появившемся окне
  • Редактирование/Удаление: Нажмите кнопку "✏️" рядом с ФИО студента в любой таблице

📊 2. Работа с оценками

  • Ввод оценок: Переключайтесь между вкладками РО-1, РО-2 и т.д. Введите оценки в ячейки (от 0 до 100)
  • Количество колонок: Измените число в поле "Оценок (колонок) в РО-X" для добавления/удаления колонок
  • Названия колонок: Нажмите "Редактировать названия" чтобы заменить "Оц.1" на реальные даты (например, "12.03")
  • Средний балл: Автоматически рассчитывается при заполнении всех колонок

🏆 3. Итоги и успеваемость

  • Просмотр итогов: Перейдите на вкладку "🏆 Итоги" для общей картины
  • Расчет семестра: Итоговый балл = среднее арифметическое всех РО
  • Буквенная оценка: Автоматически переводится по шкале от A (95-100) до F (менее 50)
  • GPA: Числовой эквивалент буквенной оценки (от 4,0 до 0)

💾 4. Сохранение и бэкапы

  • Автосохранение: Все изменения сохраняются автоматически прямо в браузере
  • Ручное сохранение: Нажмите "Сохранить" для гарантии
  • Бэкап: Нажмите "📥 Бэкап" чтобы скачать файл с данными на компьютер
  • Восстановление: Нажмите "📂 Восстановить" и выберите ранее сохраненный файл бэкапа
  • Сброс: "🗑️ Сбросить всё" - удаляет ВСЕ данные (осторожно!)

📎 5. Экспорт и печать

  • Экспорт в Excel: Нажмите "Экспорт Итогов в Excel" - создастся файл .xlsx со всеми данными
  • Печать: Нажмите Ctrl+P на вкладке "Итоги" для печати ведомости

⚙️ 6. Дополнительные возможности

  • Количество РО: Измените число в поле "РО в семестре" (до 20)
  • Фиксация ФИО: При горизонтальной прокрутке таблицы колонка с ФИО всегда остается видимой
  • Адаптивность: Журнал работает на компьютерах, планшетах и телефонах

💡 8. Советы

  • Регулярно создавайте бэкапы важных данных
  • Используйте понятные названия для колонок с оценками
  • Проверяйте итоговую таблицу перед экспортом
  • Данные хранятся в вашем браузере - не очищайте кэш без бэкапа!
