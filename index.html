📁 1. Управление группами

  • Создание группы: Нажмите "➕ Новая группа", введите название
  • Переключение групп: Выберите группу из выпадающего списка
  • Переименование: Выберите группу, нажмите "✏️ Переименовать"
  • Удаление: Выберите группу, нажмите "🗑️ Удалить группу"

📋 2. Работа со студентами

  • Загрузка списка: Нажмите "Загрузить список (.txt)"
  • Добавление вручную: Нажмите "➕ Добавить студента"
  • Редактирование: Нажмите "✏️" рядом с ФИО студента

📊 3. Оценки

  • Вводите оценки от 0 до 100 в ячейки
  • Меняйте количество колонок в РО
  • Редактируйте названия колонок (даты)
