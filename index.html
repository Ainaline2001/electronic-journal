📁 1. Управление группами

  • Создание группы: Нажмите кнопку "➕ Новая группа", введите название (например, "ИС-21", "ПД-11") и нажмите "Создать"
  • Переключение между группами: Выберите нужную группу из выпадающего списка "Группа"
  • Переименование группы: Выберите группу, нажмите "✏️ Переименовать", введите новое название
  • Удаление группы: Выберите группу, нажмите "🗑️ Удалить группу" - все данные группы будут потеряны
  • Количество студентов в группе: Отображается в скобках рядом с названием группы

📋 2. Работа со студентами

  • Загрузка списка из файла: Нажмите "📁 Загрузить список (.txt)" - файл должен содержать ФИО студентов (каждый с новой строки)
  • Добавление студента вручную: Нажмите "➕ Добавить студента", введите ФИО в появившемся окне
  • Редактирование ФИО: Нажмите кнопку "✏️" рядом с ФИО студента в любой таблице
  • Удаление студента: В режиме редактирования нажмите "🗑️ Удалить" - все оценки студента будут удалены

📊 3. Работа с оценками

  • Ввод оценок: Переключайтесь между вкладками (РО-1, РО-2 и т.д.), вводите оценки в ячейки (от 0 до 100)
  • Количество колонок: Измените число в поле "Оценок (колонок) в РО-X" - колонки добавятся или удалятся автоматически
  • Названия колонок: Нажмите "✏️ Редактировать названия" чтобы заменить "Оц.1" на реальные даты (например, "12.03", "Контрольная")
  • Средний балл по РО: Рассчитывается автоматически при заполнении всех колонок
  • Подсказка: При вводе оценки нажмите Enter для быстрого перехода к следующей ячейке

🏆 4. Итоги и успеваемость

  • Просмотр итогов: Перейдите на вкладку "🏆 Итоги" для общей картины успеваемости
  • Расчет семестра: Итоговый балл = среднее арифметическое всех РО
  • Буквенная оценка: Автоматически переводится по шкале (см. таблицу ниже)
  • GPA: Числовой эквивалент буквенной оценки (от 4,0 до 0)

💾 6. Сохранение и бэкапы

  • Автосохранение: Все изменения сохраняются автоматически прямо в браузере
  • Ручное сохранение: Нажмите "💾 Сохранить" для гарантии
  • Создание бэкапа: Нажмите "📥 Бэкап всех групп" - сохранятся ВСЕ группы в один файл
  • Восстановление из бэкапа: Нажмите "📂 Восстановить" и выберите ранее сохраненный файл
  • Сброс всех данных: "🗑️ Сбросить всё" - удаляет ВСЕ группы и оценки (осторожно!)
  • Важно: При очистке кэша браузера данные могут потеряться - регулярно делайте бэкапы!

📎 7. Экспорт и печать

  • Экспорт в Excel: Нажмите "📊 Экспорт в Excel" - создастся файл .xlsx с итоговой ведомостью
  • Имя файла: Можно ввести любое понятное название (например, "ИС-21_итоги_май")
  • Печать: Нажмите Ctrl+P на вкладке "Итоги" для печати ведомости
  • Формат файла: Экспорт поддерживает настоящий формат Excel с корректным отображением кириллицы

⚙️ 8. Дополнительные возможности

  • Количество РО: Измените число в поле "РО в семестре" (до 20 РО)
  • Фиксация ФИО: При горизонтальной прокрутке таблицы колонка с ФИО всегда остается видимой
  • Адаптивность: Журнал работает на компьютерах, планшетах и телефонах
  • Автосохранение: Все изменения мгновенно сохраняются (индикатор в правом нижнем углу)
  • Совместимость: Работает во всех современных браузерах (Chrome, Edge, Safari, Firefox)

❓ 9. Частые вопросы

  • Как перенести данные на другой компьютер? Сделайте бэкап на первом компьютере, затем восстановите на втором
  • Что делать, если пропали данные? Проверьте, не очищали ли вы кэш браузера. Восстановите из последнего бэкапа
  • Как добавить больше 20 РО? Введите число в поле "РО в семестре" (максимум 20)
  • Можно ли работать офлайн? Да! После первого открытия страница кэшируется
  • Как изменить дату оценки? Нажмите "✏️ Редактировать названия" в нужном РО

💡 10. Полезные советы

  • Регулярно создавайте бэкапы важных данных (раз в неделю)
  • Используйте понятные названия для колонок с оценками (реальные даты)
  • Проверяйте итоговую таблицу перед экспортом в Excel
  • Для быстрого ввода оценок используйте цифровую клавиатуру
  • Создавайте отдельную группу для каждого учебного класса/курса
  • Делайте бэкап перед важными изменениями в журнале
