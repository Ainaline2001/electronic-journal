📁 1. Управление группами

  • Создание группы: Нажмите "➕ Новая группа", введите название (например, "ИС-21")
  • Переключение групп: Выберите группу из выпадающего списка
  • Переименование: Выберите группу, нажмите "✏️ Переименовать"
  • Удаление: Выберите группу, нажмите "🗑️ Удалить группу"

📋 2. Работа со студентами

  • Загрузка списка: Нажмите "Загрузить список (.txt)" - файл с ФИО построчно
  • Добавление вручную: Нажмите "➕ Добавить студента" и введите ФИО
  • Редактирование/Удаление: Нажмите кнопку "✏️" рядом с ФИО студента

📊 3. Работа с оценками

  • Ввод оценок: Переключайтесь между вкладками РО-1, РО-2 и т.д.
  • Количество колонок: Измените число в поле "Оценок (колонок) в РО-X"
  • Названия колонок: Нажмите "✏️ Редактировать названия" чтобы заменить "Оц.1" на даты

🏆 4. Итоги и успеваемость

  • Итоговый балл = среднее арифметическое всех РО
  • Шкала оценок: A (95-100) → F (менее 50)
  • GPA: Числовой эквивалент от 4,0 до 0

💾 5. Сохранение и бэкапы

  • Автосохранение: Все изменения сохраняются автоматически
  • Бэкап всех групп: Сохраняет все группы в один файл
  • Восстановление: Загрузите ранее сохраненный файл бэкапа

📎 6. Экспорт

  • Нажмите "📊 Экспорт в Excel" - создастся файл .xlsx
  • Имя файла по умолчанию: "Группа_Итоговая_Ведомость_дата.xlsx"
