📁 1. Управление группами

  • Создание группы: Нажмите "➕ Новая", введите название
  • Переключение: Выберите группу из списка
  • Переименование: Нажмите "✏️" рядом с группой
  • Удаление: Нажмите "🗑️" - все данные группы будут удалены

📋 2. Работа со студентами

  • Загрузка списка: Нажмите "Загрузить список (.txt)" - файл с ФИО построчно
  • Добавление: Нажмите "➕ Добавить студента"
  • Редактирование: Нажмите "✏️" рядом с ФИО
  • Удаление: В режиме редактирования нажмите "Удалить"

🎤 3. Голосовой ввод оценок

  • Быстрый ввод: Нажмите "🎤 Быстрый ввод" и скажите: "Иванов 85"
  • В ячейке: Нажмите кнопку 🎤 в любой ячейке оценки
  • Массовый ввод: Скажите "всем 75" для всей группы
  • Поддерживаются: числа и слова (пять, десять, двадцать)

📊 4. Работа с оценками

  • Ввод: Вводите оценки от 0 до 100 в ячейки
  • Колонки: Меняйте количество колонок в РО
  • Названия: Редактируйте названия колонок (например, даты)

🏆 5. Итоги и успеваемость

  • Итоговый балл: Среднее арифметическое всех РО
  • Буквенная оценка: A (95-100) → F (менее 50)
  • GPA: Числовой эквивалент от 4,0 до 0

💾 7. Сохранение и бэкапы

  • Автосохранение: Все изменения сохраняются автоматически
  • Бэкап: Сохраняет все группы в один файл
  • Восстановление: Загрузите ранее сохраненный файл
